# Recuperbench

This is a small workbench for manufacturing parts for recuperators. 
It based on GT2560 Rev A+ and 12864KBA v.2 (previously it was a 3D-printer).

Besides default Arduino libs, the program uses the following libs:
- **[EncButton by AlexGyver](https://github.com/GyverLibs/EncButton)**
